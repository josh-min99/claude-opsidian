## 배운 것

### ItemView = 커스텀 패널
- `ItemView`를 상속해서 사이드바/탭에 뜨는 패널을 만든다.
- `getViewType()`(고유 id), `getDisplayText()`(탭 제목), `getIcon()`, `onOpen()`(그리기).
- `registerView(TYPE, leaf => new MyView(leaf, plugin))`로 등록.
- 열기: `workspace.getRightLeaf(false).setViewState({type})` → `revealLeaf`.

### createEl/createDiv (innerHTML 금지)
- Obsidian은 보안상 `innerHTML` 대신 `el.createDiv({cls, text})` 같은 헬퍼 권장.
- HTML 문자열을 파싱하지 않아 XSS 위험이 없고 타입 안전.
- 막대 채움은 `fill.style.width = `${pct}%``로 동적 제어.

### 윈도우 로직 일반화
- 5시간이든 7일이든 "블록" 분할 규칙은 동일 → `windowMs` 인자 하나로 통합.
- 코드 중복 제거 + 나중에 다른 윈도우(일별 등) 추가도 쉬움.

### 자동 갱신
- `window.setInterval`로 N초마다 `refreshViews()` 호출.
- `registerInterval`로 등록하면 플러그인 unload 시 자동 정리됨.
- 설정에서 주기 바꾸면 `restartRefreshTimer()`로 타이머 재시작.

### strict 모드가 잡아준 것
- `settings` 프로퍼티 초기화 미보장 → `settings!:`로 "나중에 할당" 명시.
- `CurrentWindow` import 출처 혼동 → 타입은 types.ts에서 직접 import.
